## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/mercado-libre-app.git
```

Go to the project directory

```bash
  cd mercado-libre-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
